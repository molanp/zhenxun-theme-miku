# zhenxun-theme-miku

适用于 zhenxun_bot 的 miku 色系主题

# 使用方法

1. 下载本仓库
2. 将 `miku` 文件夹移动到到 zhenxun_bot 目录的 `/resources/themes/` 文件夹中
3. 对你的 Bot 发送 `ui theme miku` 以启用主题
